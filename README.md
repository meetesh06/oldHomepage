The blog posts are written in markdown format and are contained inside **public/posts** directory.
The markdown files must have the following headers to be recognized.

```
<!-- Title -->
<!-- Category -->
<!-- Lineage -->
<!-- Tagline -->
<!-- Date of Creation: (DD/MM/YY) -->
```
